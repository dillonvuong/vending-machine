How to Run
1. cd into client and run the following command after installing packages (npm install): "npm run start", a browser will be open on localhost:3000
2. put the .env file that Dillon gave you into the server folder
3. cd into server and run the following command after installing packages (npm install): "npm run devStart", a local server will be run on localhost:8080
